### 최대한 명확히 작성
자유롭게 작성하되, 다른 사람이 봤을 때 쉽게 이해할 수 있어야 함
```
❌ UART 수정
⭕ UART 수신 버퍼 오버플로우 발생 시 데이터 폐기 로직 추가
```
```
❌ 타이머 설정 변경
⭕ TIM2 인터럽트 주기 1ms → 10ms로 변경
```
```
❌ 센서 코드 수정
⭕ IMU 초기화 시 가속도 범위 ±2g로 설정
```

<br>

### Commit prefix는 사용 X
```
❌ [fix] UART 수신 버퍼 오버플로우 발생 시 데이터 폐기 로직 추가
⭕ UART 수신 버퍼 오버플로우 발생 시 데이터 폐기 로직 추가
```
```
❌ [feat] SPI 기반 OLED 드라이버 추가
⭕ SPI 기반 OLED 드라이버 추가
```

<br>

### 커밋 1개 = 기능 1개
```
❌ PWM 제어 로직 개선 및 모터 방향 제어 핀 수정
⭕ PWM 듀티 계산 방식 정수 연산으로 변경
- float 연산 제거
- 0~100% → 0~1000 스케일 변환

⭕ 모터 방향 제어 GPIO 핀 번호 수정
- PB3 → PB5로 변경
```
```
❌ FreeRTOS 태스크 추가 및 스택 크기 조정
⭕ 센서 데이터 처리 태스크 추가
- 우선순위 2
- 주기 10ms

⭕ 기존 통신 태스크 스택 크기 512 → 1024로 변경
```
```
❌ ADC 설정 변경 및 배터리 전압 보정 로직 추가
⭕ ADC 샘플링 시간 15cycle → 55cycle로 변경

⭕ 배터리 전압 보정 계수 1.00 → 1.07 적용
- 실제 측정값 기준 오차 보정
```
